# Task 1: BMI Calculator

## Internship
Oasis Infobyte – Python Programming Internship

## Project Description
This project is a command-line based BMI (Body Mass Index) Calculator developed using Python.  
It takes user input for weight and height, calculates BMI, and classifies the result into health categories.

## Features
- Accepts weight in kilograms
- Accepts height in meters
- Calculates BMI using standard formula
- Categorizes BMI into:
  - Underweight
  - Normal
  - Overweight
  - Obese
- Handles invalid input using exception handling

## Concepts Used
- Functions
- Conditional statements
- Exception handling (try-except)
- User input validation
- Mathematical calculations
- Modular programming

## BMI Formula
BMI = weight / (height * height)


## How It Works
1. User enters weight and height.
2. Program validates the input.
3. BMI is calculated using the formula.
4. BMI category is determined.
5. Result is displayed to the user.
