# Task 2: Random Password Generator

## Internship
Oasis Infobyte – Python Programming Internship

## Project Description
This project is a command-line based Random Password Generator built using Python.  
The program generates secure and customizable passwords based on user-defined criteria.

## Features
- User-defined password length
- Option to include:
  - Letters (uppercase and lowercase)
  - Digits
  - Special characters
- Input validation using exception handling
- Random password generation using Python standard libraries

## Concepts Used
- Functions and return values
- Conditional statements
- Loops
- Exception handling (try-except)
- `random` module for randomness
- `string` module for character sets
- Modular programming approach

## How It Works
1. User enters desired password length.
2. User selects character types to include.
3. Program validates inputs.
4. Password is generated randomly using selected characters.
5. Final password is displayed to the user.
